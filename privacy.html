üìä Information We Collect

Local Storage Data

We store the following information locally on your device:

  • User Preferences: Your chosen reminder interval (e.g., every 10 minutes, 30 minutes, etc.)
  • Extension Settings: Configuration options for the reminder functionality
  • Last Ayah Information: Details about the most recently displayed Quran verse
  • Alarm Schedules: Timing information for scheduled reminders

No Personal Data Collection

We do not collect, store, or transmit any personal information such as:

  • Your name, email address, or contact information
  • Browsing history or website data
  • Personal files or documents
  • Location information

üåê External API Usage

Quran API (api.alquran.cloud)

Our extension fetches Quran verses from the public API at api.alquran.cloud. When making requests:

  • We only request random Quran verses and their translations
  • No personal information is sent to this API
  • We do not store or cache the API responses
  • If the API is unavailable, we use embedded fallback verses
Note: The Quran API is a third-party service. We recommend reviewing their privacy policy at alquran.cloud for information about their data practices.

üîß Extension Permissions

Our extension requests the following permissions and here's why:

Required Permissions

  • notifications: To display Quran verse reminders as desktop notifications
  • storage: To save your reminder interval preferences locally
  • alarms: To schedule periodic reminders based on your chosen interval

Host Permissions

  • https://api.alquran.cloud/*: To fetch Quran verses from the public API
Important: We only use these permissions for the stated purposes. We do not access any other websites, read your emails, or access any other personal data.

üîí Data Security

Local Storage

All your data is stored locally on your device using Chrome's extension storage API. This means:

  • Your data never leaves your device
  • We cannot access your stored preferences
  • Data is automatically encrypted by Chrome
  • Data persists only as long as the extension is installed

No Data Transmission

We do not transmit, sell, or share any user data with third parties, except for the necessary API calls to fetch Quran verses.
